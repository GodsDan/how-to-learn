## Описание

Проект реализован на чистом HTML и CSS. Файловая структура организована согласно БЭМ *Nested* Проект содержит 11 блоков, а именно:

1. Шапка.
2. Главные проблемы в обучении.
3. Техники обучения.
4. Видео на TED.
5. История Барбары Оакли.
6. Метод Фейнмана.
7. Цифры и факты.
8. Весь мир - школа.
9. Принципы обучения.
10. Полезные ресурсы.
11. Подвал.
